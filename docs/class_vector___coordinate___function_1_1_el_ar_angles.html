Detailed Description

Provides with the angles for the robot arm.

Using Inverse Kinematics (IK) the angle for the arm with respect to the horizontal can be found as well as finding the angle required for the elbow in order reach the end-effector position described by the coordinates x and y Furthur, help/calculations were obtained from the following MATLAB documentation : https://www.mathworks.com/help/symbolic/derive-and-apply-inverse-kinematics-to-robot-arm.html
